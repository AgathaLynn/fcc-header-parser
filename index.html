Header Parser Microservice

About this project

This project is my implementation of the Request Header Parser Microservice API challenge. For this project, I used Node.js and the Express.js framework to build a simple application that gets information about the user from the html request header and returns that information in JSON.

The github repository for this project is https://github.com/AgathaLynn/fcc-header-parser.

How to use the API

To see information about yourself - or at least about how websites see you - go to http://fcc-header-parser-agathalynn.c9users.io/api. What you'll learn:

  • IP Address: Your IP address.
  • Language: Your first-choice language.
  • Language-Full: More complete information about your language preferences (Higher quality translates to greater preference).
  • Software: Information about your operating system and browser.
